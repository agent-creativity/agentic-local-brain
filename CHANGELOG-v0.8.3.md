# Changelog - v0.8.3 备份设置重构

## 概述
本次更新重构了备份管理功能，简化了用户界面，增强了云存储支持，并添加了分页功能。

## 主要变更

### 1. 备份管理页面简化
- **移除冗余字段**：删除了备份描述和云存储选择下拉框
- **简化操作流程**：备份创建时自动使用系统配置中的默认云存储设置
- **优化用户体验**：添加"自动备份配置"按钮，直接跳转到系统配置页面

### 2. 云存储配置集中化
- **新增配置界面**：在系统配置页面添加"备份配置"标签页
- **支持多种云存储**：
  - 阿里云 OSS
  - AWS S3
  - 本地存储
- **配置项**：
  - 云存储提供商选择
  - Access Key ID / Access Key
  - Secret Access Key / Secret Key
  - Bucket 名称
  - Endpoint（可选）
  - 区域（可选）

### 3. 备份列表增强
- **统一显示**：同时显示本地备份和云存储备份
- **云存储集成**：
  - 自动列出 OSS/S3 中的备份文件
  - 显示云存储位置（如 `oss://bucket/path`）
- **分页功能**：
  - 备份历史：每页显示 20 个备份
  - 备份任务：每页显示 20 个任务
  - 分页控件：`‹ 当前页 / 总页数 ›`

### 4. 删除功能改进
- **移除确认对话框**：简化删除流程，直接删除并显示提示
- **错误处理增强**：
  - 显示详细的错误信息
  - 区分本地备份和云存储备份的删除状态
- **已知限制**：目前只能删除本地元数据中的备份，云存储删除功能待实现

### 5. API 改进

#### `/api/backup/list` (GET)
- 返回本地备份和云存储备份的合并列表
- 每个备份包含 `id` 字段（用于删除操作）
- 云存储备份显示 `cloud_location` 字段

#### `/api/backup/delete/<backup_id>` (DELETE)
- 支持通过 `id` 或 `filename` 删除备份
- 返回详细的错误信息（如 "Backup not found"）

#### `/api/settings/backup` (GET/POST)
- 获取和保存备份配置
- 支持云存储凭证配置
- 保存后自动重启备份调度器

### 6. 前端改进
- **Vue 响应式分页**：使用计算属性实现高效分页
- **状态管理**：添加分页状态（`backupPage`, `backupTaskPage`）
- **用户反馈**：使用 Toast 提示显示操作结果

## 技术细节

### 文件变更
- `kb/web/static/index.html` - 备份管理页面和分页功能
- `kb/web/static/js/pages/settings.js` - 备份配置界面
- `kb/web/routes/backup.py` - 备份 API 路由
- `kb/commands/backup.py` - 备份命令和云存储集成

### 新增功能
1. 云存储备份列表查询（OSS/S3）
2. 备份配置保存和加载
3. 分页计算属性和方法
4. 备份删除 API 改进

### 已知问题和限制
1. **云存储删除**：只存在于云存储的备份无法删除（需要实现云存储删除功能）
2. **备份恢复**：云存储备份的恢复功能待实现
3. **权限管理**：云存储凭证存储在配置文件中，需要注意安全性
