Agentic Local Brain

Your personal knowledge management system — collect, process, and intelligently retrieve knowledge from multiple sources, powered by your AI coding agent.

What is Agentic Local Brain?

Agentic Local Brain is a comprehensive local-first knowledge management system designed for developers and knowledge workers. It acts as an extended brain that:

  • Collects knowledge from diverse sources: files (PDF, Markdown, text), webpages, bookmarks, academic papers, emails, and notes
  • Processes content intelligently: auto-extracts tags and summaries using a 3-tier strategy (user-provided → LLM → built-in algorithms)
  • Stores data locally with dual storage: SQLite for metadata/tags and ChromaDB for vector embeddings
  • Retrieves information through keyword search, semantic search, and RAG-based Q&A
  • Integrates with your AI coding agent via Skills for seamless knowledge capture during work

Quick Start

Get up and running in minutes:

  1. Install: curl -fsSL http://localbrain.oss-cn-shanghai.aliyuncs.com/python_installer/install.sh | sh
  2. Initialize: localbrain init
  3. Collect: localbrain collect note add "My first knowledge item" --tags "test"
  4. Search: localbrain search keyword "first knowledge"
  5. Web UI: localbrain web then visit http://127.0.0.1:11201

什么是 Agentic Local Brain？

Agentic Local Brain 是一个专为开发者和知识工作者设计的全面本地优先知识管理系统。它作为你的"第二大脑"：

  • 收集来自多种来源的知识：文件（PDF、Markdown、文本）、网页、书签、学术论文、邮件和笔记
  • 处理内容智能化：使用三层策略（用户提供 → LLM → 内置算法）自动提取标签和摘要
  • 存储数据在本地：SQLite 存储元数据/标签，ChromaDB 存储向量嵌入
  • 检索信息：通过关键词搜索、语义搜索和基于 RAG 的问答
  • 集成你的 AI 编程助手：通过 Skills 在工作中无缝捕获知识
