Skill Installation Guide

Skills are the bridge between your AI coding agent and Agentic Local Brain. By installing the knowledge collection skill, your agent can seamlessly save important information — code snippets, articles, bookmarks, insights — directly into your personal knowledge base while you work.

Why use Skills?

Instead of manually copying and pasting information into your knowledge base, your AI agent can do it automatically. Just tell it what to save, and it handles the rest — tagging, embedding, and indexing for later retrieval.

Quickest Way to Install (30 seconds)

The fastest way to install the knowledge collection skill:

Simply paste this message into your AI agent's chat window:

Help me install or update this knowledge collection skill: http://127.0.0.1:8080/docs/skills/knowledge-collect-localbrain/SKILL.md

That's it! Your agent will automatically download and install the skill. This works with Qoder, OpenClaw, Copaw, and most AI coding agents that support skill installation.

Skill Resources

  • 📄 View SKILL.md — The skill definition file (view online)
  • 📦 Download SKILL.md — Download for manual installation

The skill consists of a single SKILL.md file — no additional scripts or dependencies required.

What is a Skill?

  • A Skill is a plugin/extension that enhances your AI coding agent with new capabilities
  • The Agentic Local Brain Skill (knowledge-collect-localbrain) enables your agent to collect knowledge from your daily workflow
  • When installed, you can simply tell your agent things like "save this article to my knowledge base" or "collect this code snippet" and it will be automatically processed and stored

The skill connects to your local Agentic Local Brain instance through its LocalBrain CLI, enabling seamless knowledge management without leaving your coding environment.

Prerequisites

Before installing the skill, ensure the following requirements are met:

  1. Agentic Local Brain must be installed and initialized 
    localbrain init
  2. The web server should be running 
    localbrain web -b

    The -b flag runs the server in background mode.

  3. Verify your setup is working 
    localbrain doctor

    This command checks your configuration and connectivity.

Important

The web server must be running for the skill to function. If you restart your computer, you may need to start it again with localbrain web -b.

Installation for Qoder (QoderWork)

Automatic Installation (Recommended)

  1. Open Qoder
  2. Open the Skill Store:
    • Press Cmd+Shift+P (macOS) or Ctrl+Shift+P (Windows/Linux)
    • Type "Install Skill" and select the option
  3. Search for knowledge-collect-localbrain
  4. Click "Install"
  5. The skill will be automatically configured and ready to use
Tip

After installation, you can verify the skill is loaded by asking Qoder: "What skills do you have available?"

Manual Installation

  1. Navigate to your Qoder skills directory: 
    cd ~/.qoder/skills/
  2. Create the skill directory: 
    mkdir -p knowledge-collect-localbrain
cd knowledge-collect-localbrain
  3. Copy the skill manifest file from the Agentic Local Brain project: 
    cp /path/to/agentic-local-brain/.qoder/skills/knowledge-collect-localbrain/skill.md ./
  4. Restart Qoder to load the new skill

Installation for OpenClaw

Via Skill Marketplace

  1. Open OpenClaw
  2. Navigate to Settings → Skills/Plugins
  3. Search for Agentic Local Brain or knowledge-collect
  4. Click Install and follow the prompts
  5. Configure the connection URL (default: http://127.0.0.1:8080)

Manual Installation

  1. Locate OpenClaw's skill directory (typically ~/.openclaw/skills/) 
    cd ~/.openclaw/skills/
  2. Copy the skill manifest file: 
    mkdir -p knowledge-collect-localbrain
cp /path/to/agentic-local-brain/.qoder/skills/knowledge-collect-localbrain/skill.md ~/.openclaw/skills/knowledge-collect-localbrain/
  3. Configure the connection to your local brain instance in OpenClaw's settings
  4. Restart OpenClaw

Installation for Copaw

Via Plugin System

  1. Open Copaw
  2. Go to Settings → Extensions → Skill Store
  3. Search for knowledge-collect-localbrain
  4. Click Install and Enable
  5. Verify the connection in settings

  1. Find Copaw's plugin directory:

cd %USERPROFILE%\.copaw\plugins\

# Copy skill files to this directory

  3. Restart Windsurf

## Available Tools

### Knowledge Collection (LocalBrain)

You have access to a knowledge collection system. Use these commands:

- To save a note: `localbrain collect note add "content here" --title "Title"`
- To save a webpage: `localbrain collect webpage add `
- To save a file: `localbrain collect file `
- To search: `localbrain search semantic "query"`

The web API is available at http://127.0.0.1:8080/api/

Generic Agent Integration

For any coding agent that supports custom tools, skills, or plugins:

  1. Ensure localbrain CLI is in your PATH 
    which localbrain  # Should return the path
# If not, add to PATH or create symlink
  2. The agent needs to be able to call CLI commands or HTTP API
  3. API endpoint: http://127.0.0.1:8080/api/ (when web server is running)

Key API endpoints for skill integration:

Endpoint Method Description
/api/search/semantic POST Semantic search across your knowledge base
/api/search/keyword POST Keyword-based search
/api/rag POST RAG query for question answering
/api/items GET List all knowledge items
/api/tags GET List all tags

Collect via CLI commands:

# Add a note
localbrain collect note add "Your note content here" --title "Note Title"

# Collect a webpage
localbrain collect webpage add https://example.com/article

# Collect a file
localbrain collect file /path/to/document.pdf

# Import bookmarks
localbrain bookmark collect

Verify Installation

After installing the skill, verify it's working correctly:

  1. Open your coding agent
  2. Try a simple knowledge collection:
    "Save a note to my knowledge base: Test skill installation"
  3. Verify via CLI: 
    localbrain search keyword "Test skill installation"
  4. Or check the web UI:

    Open http://127.0.0.1:8080 in your browser

Success!

If you can see your test note in the search results or web UI, the skill is working correctly. You can now use natural language commands to manage your knowledge base.

Troubleshooting

Skill not found in marketplace

  • Ensure you're using a compatible agent version
  • Check if your agent supports the skill format
  • Try the manual installation method instead
  • Verify your internet connection if using an online marketplace

Connection refused

  • Verify localbrain web server is running: 
    localbrain web --status
  • Start the server if not running: 
    localbrain web -b
  • Check if port 8080 is already in use: 
    # macOS/Linux
lsof -i :8080

# Kill process if needed
kill -9 <PID>

Permission denied

  • Check file permissions on the skill directory: 
    ls -la ~/.qoder/skills/knowledge-collect-localbrain/
  • Fix permissions if needed: 
    # macOS/Linux
chmod -R 755 ~/.qoder/skills/knowledge-collect-localbrain/

# Ensure localbrain is executable
chmod +x ~/.localbrain/bin/localbrain

Knowledge not being saved

  1. Check localbrain doctor for configuration issues: 
    localbrain doctor
  2. Verify API key is configured: 
    localbrain config show
  3. Check logs for errors: 
    # View recent logs
tail -f ~/.localbrain/logs/localbrain.log
  4. Ensure the embedding service is configured correctly: 
    # Test embedding service
localbrain test embedding

Agent can't find the skill

  • Restart your coding agent after installation
  • Check the skill directory path is correct
  • Verify the skill manifest file exists and is valid
  • Check your agent's documentation for skill installation requirements

常见问题排查

在市场中找不到 Skill

  • 确保您使用的是兼容的助手版本
  • 检查您的助手是否支持该 Skill 格式
  • 尝试使用手动安装方法
  • 如果使用在线市场，请检查网络连接

连接被拒绝

  • 验证 localbrain web 服务器是否正在运行： 
    localbrain web --status
  • 如果未运行，启动服务器： 
    localbrain web -b
  • 检查端口 8080 是否被占用： 
    # macOS/Linux
lsof -i :8080

# 如需终止进程
kill -9 <PID>

权限被拒绝

  • 检查 Skill 目录的文件权限： 
    ls -la ~/.qoder/skills/knowledge-collect-localbrain/
  • 如需修复权限： 
    # macOS/Linux
chmod -R 755 ~/.qoder/skills/knowledge-collect-localbrain/

# 确保 localbrain 可执行
chmod +x ~/.localbrain/bin/localbrain

知识未保存

  1. 检查 localbrain doctor 查看配置问题： 
    localbrain doctor
  2. 验证 API key 是否已配置： 
    localbrain config show
  3. 检查日志中的错误： 
    # 查看最近的日志
tail -f ~/.localbrain/logs/localbrain.log
  4. 确保嵌入服务配置正确： 
    # 测试嵌入服务
localbrain test embedding

助手找不到 Skill

  • 安装后重启您的编程助手
  • 检查 Skill 目录路径是否正确
  • 验证 Skill 配置文件是否存在且有效
  • 查阅您助手的文档了解 Skill 安装要求
