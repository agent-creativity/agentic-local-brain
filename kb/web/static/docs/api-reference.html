API Reference

The REST API is served by FastAPI at http://127.0.0.1:11201 (default). Interactive API docs at http://127.0.0.1:11201/api/docs.

Dashboard

GET /api/dashboard/stats

Returns knowledge base statistics.

Response example:

{
  "total_items": 42,
  "total_tags": 15,
  "items_by_type": {"note": 10, "webpage": 12, "bookmark": 8, "file": 5, "paper": 4, "email": 3}
}

GET /api/dashboard/recent

Returns recent items by type.

Items

GET /api/items

List items with pagination and filtering.

Query params: page, per_page, type, tag, q (search)

GET /api/items/{id}

Get item by ID. Returns full item details with content.

PUT /api/items/{id}

Update item metadata (title, tags, summary).

DELETE /api/items/{id}

Delete a knowledge item.

Tags

GET /api/tags

List all tags with item counts.

GET /api/tags/{name}/items

Get items associated with a specific tag.

POST /api/tags/merge

Merge two tags. Body: {"source": "old-tag", "target": "new-tag"}

DELETE /api/tags/{name}

Delete a tag.

Search

POST /api/search/keyword (or GET /api/search?q=...)

Keyword search. Body: {"query": "search terms", "limit": 10}

POST /api/search/semantic

Semantic vector search. Body: {"query": "conceptual question", "limit": 10}

Requires embedding service.

POST /api/rag

RAG query. Body: {"question": "What is X?", "limit": 5}

Requires embedding + LLM services.

Response: {"answer": "...", "sources": [...]}

Enhanced RAG (v0.7)

POST /api/rag/chat

Multi-turn conversational RAG with context preservation.

Request body:

{
  "query": "What is machine learning?",
  "session_id": "optional-session-id",
  "use_graph": true,
  "use_topics": true
}

Response includes conversation history and source citations.

GET /api/rag/conversations

List all conversation sessions.

Returns: [{"session_id": "...", "created_at": "...", "message_count": 5}]

GET /api/rag/conversations/{session_id}

Get full conversation history for a session.

Returns complete conversation with all messages and sources.

DELETE /api/rag/conversations/{session_id}

Delete a specific conversation session.

POST /api/rag/suggest

Get query suggestions based on knowledge base content.

Request body: {"query": "machine learning"}

Returns suggested follow-up questions.

GET /api/dashboard/rag-stats

Get RAG usage analytics and statistics.

Returns query counts, response times, and source usage metrics.

Wiki API (v0.7)

GET /api/wiki/tree

Get wiki structure as hierarchical navigation tree.

Returns topics with nested categories and article counts.

{
  "topics": [
    {
      "topic_id": "cluster_001",
      "label": "Machine Learning",
      "categories": [
        {"category_id": "...", "name": "Deep Learning", "article_count": 5}
      ]
    }
  ],
  "stats": {"total_articles": 42, "total_categories": 8}
}

GET /api/wiki/categories/{category_id}/articles

Get articles belonging to a specific category.

GET /api/wiki/topics/{topic_id}/articles

Get all articles for a given topic across all its categories.

GET /api/wiki/articles

List wiki articles with filtering.

Query params:

  • article_type — Filter by type: topic, entity
  • limit — Max results (default: 100, max: 500)
  • offset — Offset for pagination (default: 0)

GET /api/wiki/articles/{article_id}

Get article content with metadata.

Returns full article content (Markdown), metadata, and source documents.

{
  "article_id": "ml-intro",
  "title": "Introduction to Machine Learning",
  "article_type": "topic",
  "content": "# Introduction...",
  "word_count": 1500,
  "source_doc_ids": ["doc1", "doc2"],
  "entity_refs": ["neural-networks", "supervised-learning"],
  "compiled_at": "2026-04-10T12:00:00Z",
  "version": 2
}

GET /api/wiki/search?q={query}

Search wiki articles by title.

Query params: q (required), limit (default: 20, max: 100)

GET /api/wiki/entities

List all entities with wiki cards.

GET /api/wiki/entities/{entity_id}

Get entity details and associated wiki card.

GET /api/wiki/stats

Get wiki statistics (article counts, category counts, etc.).

Error Handling

Standard HTTP status codes. Error response format: {"detail": "Error message"}

Knowledge Mining (v0.6)

GET /api/graph

Returns knowledge graph data for visualization (nodes and edges).

GET /api/knowledge/{id}/related

Returns related documents for a given knowledge item, based on embedding similarity and shared entities.

GET /api/topics

Returns all topic clusters with document counts and labels.

GET /api/topics/{id}/documents

Returns documents belonging to a specific topic cluster.

GET /api/topics/trend

Returns topic trend data for time-based visualization.

GET /api/recommendations

Returns personalized reading recommendations based on user patterns. Response includes recommendation reasons.

API 参考

REST API 由 FastAPI 在 http://127.0.0.1:11201（默认）提供服务。交互式 API 文档位于 http://127.0.0.1:11201/api/docs。

仪表盘

GET /api/dashboard/stats

返回知识库统计信息。

响应示例：

{
  "total_items": 42,
  "total_tags": 15,
  "items_by_type": {"note": 10, "webpage": 12, "bookmark": 8, "file": 5, "paper": 4, "email": 3}
}

GET /api/dashboard/recent

按类型返回最近的项目。

项目

GET /api/items

列出项目，支持分页和过滤。

查询参数：page、per_page、type、tag、q（搜索）

GET /api/items/{id}

通过 ID 获取项目。返回完整的项目详情和内容。

PUT /api/items/{id}

更新项目元数据（标题、标签、摘要）。

DELETE /api/items/{id}

删除知识项目。

标签

GET /api/tags

列出所有标签及其项目数量。

GET /api/tags/{name}/items

获取与特定标签关联的项目。

POST /api/tags/merge

合并两个标签。请求体：{"source": "old-tag", "target": "new-tag"}

DELETE /api/tags/{name}

删除标签。

搜索

POST /api/search/keyword（或 GET /api/search?q=...）

关键词搜索。请求体：{"query": "search terms", "limit": 10}

POST /api/search/semantic

语义向量搜索。请求体：{"query": "conceptual question", "limit": 10}

需要嵌入服务。

POST /api/rag

RAG 查询。请求体：{"question": "What is X?", "limit": 5}

需要嵌入 + LLM 服务。

响应：{"answer": "...", "sources": [...]}

增强 RAG (v0.7)

POST /api/rag/chat

多轮对话式 RAG，支持上下文保持。

请求体：

{
  "query": "什么是机器学习？",
  "session_id": "可选会话ID",
  "use_graph": true,
  "use_topics": true
}

响应包含对话历史和来源引用。

GET /api/rag/conversations

列出所有会话。

返回：[{"session_id": "...", "created_at": "...", "message_count": 5}]

GET /api/rag/conversations/{session_id}

获取指定会话的完整对话历史。

返回包含所有消息和来源的完整对话。

DELETE /api/rag/conversations/{session_id}

删除指定的会话。

POST /api/rag/suggest

基于知识库内容获取查询建议。

请求体：{"query": "machine learning"}

返回建议的后续问题。

GET /api/dashboard/rag-stats

获取 RAG 使用分析和统计信息。

返回查询次数、响应时间和来源使用指标。

Wiki API (v0.7)

GET /api/wiki/tree

获取 Wiki 结构作为分层导航树。

返回包含嵌套分类和文章数的主题列表。

{
  "topics": [
    {
      "topic_id": "cluster_001",
      "label": "机器学习",
      "categories": [
        {"category_id": "...", "name": "深度学习", "article_count": 5}
      ]
    }
  ],
  "stats": {"total_articles": 42, "total_categories": 8}
}

GET /api/wiki/categories/{category_id}/articles

获取指定分类下的文章。

GET /api/wiki/topics/{topic_id}/articles

获取指定主题下的所有文章（跨所有分类）。

GET /api/wiki/articles

列出 Wiki 文章，支持过滤。

查询参数：

  • article_type — 按类型过滤：topic、entity
  • limit — 最大结果数（默认：100，最大：500）
  • offset — 分页偏移量（默认：0）

GET /api/wiki/articles/{article_id}

获取文章内容和元数据。

返回完整文章内容（Markdown）、元数据和源文档。

{
  "article_id": "ml-intro",
  "title": "机器学习简介",
  "article_type": "topic",
  "content": "# 简介...",
  "word_count": 1500,
  "source_doc_ids": ["doc1", "doc2"],
  "entity_refs": ["neural-networks", "supervised-learning"],
  "compiled_at": "2026-04-10T12:00:00Z",
  "version": 2
}

GET /api/wiki/search?q={query}

按标题搜索 Wiki 文章。

查询参数：q（必需），limit（默认：20，最大：100）

GET /api/wiki/entities

列出所有带有 Wiki 卡片的实体。

GET /api/wiki/entities/{entity_id}

获取实体详情和关联的 Wiki 卡片。

GET /api/wiki/stats

获取 Wiki 统计信息（文章数、分类数等）。

错误处理

标准 HTTP 状态码。错误响应格式：{"detail": "Error message"}

知识挖掘 (v0.6)

GET /api/graph

返回知识图谱可视化数据（节点和边）。

GET /api/knowledge/{id}/related

返回指定知识条目的相关文档，基于嵌入相似度和共享实体。

GET /api/topics

返回所有主题聚类，包含文档数量和标签。

GET /api/topics/{id}/documents

返回特定主题聚类下的文档。

GET /api/topics/trend

返回主题趋势数据，用于基于时间的可视化。

GET /api/recommendations

返回基于用户模式的个性化阅读推荐，包含推荐理由。
