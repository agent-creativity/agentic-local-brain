Contact & Collaboration

About the Project

Agentic Local Brain is developed as an open-source personal knowledge management tool, designed to empower developers with an AI-enhanced local knowledge system.

Maintainer

  • YunZhi (云智)
  • Email: yunzhi.xdl@alibaba-inc.com

How to Contribute

We welcome contributions of all kinds:

  • Bug reports and feature requests
  • Code contributions (new collectors, providers, UI improvements)
  • Documentation improvements
  • Translations
  • Testing and feedback

Development Setup

git clone <repository-url>
cd agentic-local-brain
pip install -e .
pytest  # Run tests

Code Style

  • Formatter: Black (line length 88)
  • Import sorting: isort
  • Type checking: mypy
  • Docstrings: Google style

Reporting Issues

When reporting bugs, please include:

  1. Your OS and Python version
  2. Output of localbrain doctor
  3. Steps to reproduce the issue
  4. Expected vs actual behavior
  5. Relevant log output from ~/.localbrain/logs/
