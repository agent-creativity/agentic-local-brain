CLI Reference

The CLI uses an object-first (noun-verb) pattern. Primary command: localbrain (alias: kb).

Collection Commands

All collection commands support:

  • --tags, -t — Provide tags manually (multiple allowed)
  • --summary, -s — Provide a summary manually
  • --auto-extract / --no-auto-extract — Toggle auto-extraction (default: enabled)
  • --skip-existing — Skip if already collected

File Collection

localbrain collect file add <path>
localbrain collect file add ~/documents/paper.pdf
localbrain collect file add ~/notes/ideas.md --tags "ideas"

Webpage Collection

localbrain collect webpage add <url>
localbrain collect webpage add https://example.com/article --tags "reference"

Paper Collection

localbrain collect paper add <source>
localbrain collect paper add arxiv:2401.12345
localbrain collect paper add https://arxiv.org/abs/2401.12345

Email Collection

localbrain collect email add <path>
localbrain collect email add ~/emails/message.eml
localbrain collect email add ~/emails/archive.mbox

Bookmark Collection

localbrain collect bookmark add <url> --tags "tag1"
localbrain collect bookmark import --browser chrome
localbrain collect bookmark import --browser firefox
localbrain collect bookmark import --file bookmarks.html

Note Collection

localbrain collect note add "Your note text" --tags "tag1,tag2" --summary "Brief summary"

Search Commands

localbrain search semantic <query>    # Vector-based semantic search
localbrain search keyword <keywords>  # Full-text keyword search
localbrain search rag <question>      # RAG-based Q&A
localbrain search tags -t <tag>       # Find items by tag

Tag Management

localbrain tag list                    # List all tags
localbrain tag merge <source> <target> # Merge tags
localbrain tag delete <name>           # Delete a tag

System Commands

Command Description
localbrain init Initialize knowledge base
localbrain init setup Guided setup wizard
localbrain config show Display configuration
localbrain stats Show statistics
localbrain doctor System diagnostics
localbrain export Export knowledge base
localbrain --version Show version

Web Server

localbrain web              # Start foreground
localbrain web -b           # Start background (daemon)
localbrain web --status     # Check status
localbrain web --stop       # Stop background server
localbrain web --port 9090  # Custom port

Maintenance

localbrain self-update           # Update to latest
localbrain self-update --check   # Check for updates
localbrain self-update --rollback # Revert to previous
localbrain uninstall             # Remove (preserves data)

Testing

localbrain test embedding  # Test embedding service
localbrain test llm        # Test LLM service

Knowledge Mining

localbrain mine run                # Run all mining tasks (graph, relations, topics, recommendations)
localbrain graph rebuild           # Rebuild knowledge graph
localbrain graph stats             # Show graph statistics
localbrain topics rebuild          # Rebuild topic clusters
localbrain topics list             # List all topic clusters

LLM Wiki (v0.7)

localbrain wiki compile            # Compile wiki articles from topic clusters
localbrain wiki compile --force    # Regenerate all articles, ignoring staleness
localbrain wiki compile --stale-only  # Only recompile changed topics (default)
localbrain wiki list               # List compiled articles (hierarchical view)
localbrain wiki list --flat        # Show flat list instead of tree
localbrain wiki list --type topic  # Filter by type: topic, entity
localbrain wiki show <slug>        # Display article content in terminal

RAG Commands

localbrain search rag "your question"     # Single-turn RAG query
localbrain search rag "question" --chat   # Start multi-turn conversation

For multi-turn conversational RAG with context preservation, use the Web API at /api/rag/chat.

CLI 参考

CLI 使用对象优先（名词-动词）模式。主命令：localbrain（别名：kb）。

收集命令

所有收集命令都支持：

  • --tags, -t — 手动提供标签（可多次使用）
  • --summary, -s — 手动提供摘要
  • --auto-extract / --no-auto-extract — 切换自动提取（默认：启用）
  • --skip-existing — 如果已收集则跳过

文件收集

localbrain collect file add <path>
localbrain collect file add ~/documents/paper.pdf
localbrain collect file add ~/notes/ideas.md --tags "ideas"

网页收集

localbrain collect webpage add <url>
localbrain collect webpage add https://example.com/article --tags "reference"

论文收集

localbrain collect paper add <source>
localbrain collect paper add arxiv:2401.12345
localbrain collect paper add https://arxiv.org/abs/2401.12345

邮件收集

localbrain collect email add <path>
localbrain collect email add ~/emails/message.eml
localbrain collect email add ~/emails/archive.mbox

书签收集

localbrain collect bookmark add <url> --tags "tag1"
localbrain collect bookmark import --browser chrome
localbrain collect bookmark import --browser firefox
localbrain collect bookmark import --file bookmarks.html

笔记收集

localbrain search rag <question>      # 基于 RAG 的问答

localbrain wiki compile            # 从主题聚类编译 Wiki 文章

localbrain wiki compile --stale-only  # 仅重新编译已更改的主题（默认）

如需使用支持上下文保持的多轮对话 RAG，请使用 Web API /api/rag/chat。
