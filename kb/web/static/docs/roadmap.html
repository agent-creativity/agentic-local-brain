Roadmap

Current Version (v0.7.x)

Everything from v0.6.x plus Enhanced RAG and LLM Wiki capabilities:

Core Features

  • Multi-source collection (6 collector types)
  • Smart 3-tier tag & summary extraction
  • Keyword, semantic, and RAG search
  • CLI + Web interface
  • Cross-platform installation (Python package, binary)
  • Agent Skill integration
  • Graceful degradation without AI services
  • Background web server mode

Knowledge Mining (v0.6)

  • Automatic knowledge graph construction — entity extraction and relationship visualization
  • Cross-document relationship discovery — find connections via embedding similarity and shared entities
  • Topic clustering and trend analysis — HDBSCAN clustering with LLM-generated labels
  • Smart recommendations based on reading patterns — time-decay weighted, with reason generation
  • Batch mining CLI: localbrain mine run

Enhanced RAG (v0.7.0)

  • Multi-stage retrieval pipeline — query expansion, hybrid retrieval, LLM reranking
  • Multi-turn conversational Q&A with context preservation
  • Token-aware context assembly with hierarchical formatting
  • Entity and topic enrichment for better context
  • Conversation management with session persistence
  • Query suggestions and RAG analytics dashboard

LLM Wiki (v0.7.1 - v0.7.3)

  • Automatic wiki article compilation from topic clusters
  • Topic articles with hierarchical category organization
  • Entity summary cards for key concepts
  • Wiki-link cross-references between articles
  • Tree-based navigation in web UI
  • Wiki search and related article discovery

Long-term Vision

Agentic Local Brain aims to become the universal personal knowledge layer for developers — a local-first, privacy-preserving system that seamlessly integrates with your daily workflow through AI agents, making your accumulated knowledge always accessible and actionable.

当前版本 (v0.7.x)

知识挖掘（v0.6）

  • 自动知识图谱构建 — 实体抽取与关系可视化

  • 多阶段检索流水线 — 查询扩展、混合检索、LLM 重排序
  • 多轮对话式问答，支持上下文保持
  • Token 感知上下文组装，支持分层格式
  • 实体和主题增强，提供更好的上下文
  • 会话管理，支持会话持久化
  • 查询建议和 RAG 分析仪表盘

LLM Wiki (v0.7.1 - v0.7.3)

  • 从主题聚类自动生成 Wiki 文章
  • 主题文章支持分层分类组织
  • 关键概念实体摘要卡片
  • 文章间 Wiki 链接交叉引用
  • Web 界面树形导航
  • Wiki 搜索和相关文章发现
