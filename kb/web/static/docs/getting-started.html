Getting Started

Get up and running with Agentic Local Brain in under 5 minutes.

What is Agentic Local Brain?

Agentic Local Brain is a local-first personal knowledge management system for developers and knowledge workers. It collects information from multiple sources (files, webpages, papers, bookmarks, emails, notes), processes them with intelligent tagging and embedding, and provides powerful search capabilities including semantic search and RAG-based Q&A. It integrates with AI coding agents via Skills for seamless knowledge capture.

Installation

Option 1: Python Package (Recommended)

Works on all platforms without security warnings. Requires Python 3.8+.

macOS/Linux:

curl -fsSL https://localbrain.io.alibaba-inc.com/python_installer/install.sh | sh

Windows (PowerShell):

irm https://localbrain.io.alibaba-inc.com/python_installer/install.ps1 | iex

The installer will: check Python 3.8+, create a virtual environment at ~/.localbrain/venv, download and install the wheel package, add localbrain to your PATH.

Option 2: Binary Install (No Python Required)

For systems without Python. Standalone binary.

macOS/Linux:

curl -fsSL https://localbrain.io.alibaba-inc.com/binary_installer/install.sh | sh

Windows (PowerShell):

irm https://localbrain.io.alibaba-inc.com/binary_installer/install.ps1 | iex
⚠️ Warning

macOS binary requires Gatekeeper bypass: xattr -cr ~/.localbrain/bin/localbrain

Option 3: Install from Source

git clone <repository-url>
cd agentic-local-brain
pip install -e .
localbrain --version

Post-Installation Setup

localbrain doctor      # Verify installation
localbrain init setup  # Initialize with guided setup

The init command creates:

  • Configuration at ~/.localbrain/config.yaml
  • Data directory at ~/.knowledge-base/
  • SQLite database for metadata
  • ChromaDB storage for vector embeddings

Configure AI Services (Optional)

For intelligent tag extraction and semantic search, configure an AI provider:

localbrain config show  # View current config

Edit ~/.localbrain/config.yaml:

embedding:
  provider: dashscope
  dashscope:
    model: text-embedding-v4
    api_key: your-api-key-here
llm:
  provider: dashscope
  model: qwen3.5-plus
  api_key: your-api-key-here

Test connectivity:

localbrain test embedding
localbrain test llm
ℹ️ Info

AI services are optional. Without them, Local Brain still works using built-in TF-IDF and extractive algorithms.

Collect Your First Knowledge

Add a Note

localbrain collect note add "Docker containers share the host OS kernel" --tags "docker,devops"

Add a Webpage

localbrain collect webpage add https://docs.python.org/3/tutorial/

Add a Local File

localbrain collect file add ~/Documents/my-notes.md

Import Browser Bookmarks

localbrain collect bookmark import --browser chrome

Search Your Knowledge

localbrain search keyword "docker"
localbrain search semantic "how do containers work"
localbrain search rag "What are the benefits of containerization?"

Start the Web Interface

localbrain web           # Foreground mode
localbrain web -b        # Background mode (daemon)
localbrain web --status  # Check status
localbrain web --stop    # Stop background server

Visit http://127.0.0.1:8080 to see your knowledge base.

什么是 Agentic Local Brain？

Agentic Local Brain 是一个本地优先的个人知识管理系统，专为开发者和知识工作者设计。它可以从多种来源收集信息（文件、网页、论文、书签、邮件、笔记），通过智能标签和嵌入进行处理，并提供强大的搜索功能，包括语义搜索和基于 RAG 的问答。它通过 Skills 与 AI 编程助手集成，实现无缝的知识捕获。

安装

选项 1：Python 包（推荐）

适用于所有平台，无安全警告。需要 Python 3.8+。

macOS/Linux：

curl -fsSL https://localbrain.io.alibaba-inc.com/python_installer/install.sh | sh

Windows (PowerShell)：

irm https://localbrain.io.alibaba-inc.com/python_installer/install.ps1 | iex

安装程序将：检查 Python 3.8+、在 ~/.localbrain/venv 创建虚拟环境、下载并安装 wheel 包、将 localbrain 添加到 PATH。

选项 2：二进制安装（无需 Python）

适用于没有 Python 的系统。独立二进制文件。

macOS/Linux：

curl -fsSL https://localbrain.io.alibaba-inc.com/binary_installer/install.sh | sh

Windows (PowerShell)：

irm https://localbrain.io.alibaba-inc.com/binary_installer/install.ps1 | iex
⚠️ 警告

macOS 二进制文件需要绕过 Gatekeeper：xattr -cr ~/.localbrain/bin/localbrain

选项 3：从源码安装

git clone <repository-url>
cd agentic-local-brain
pip install -e .
localbrain --version

安装后设置

localbrain doctor      # 验证安装
localbrain init setup  # 使用引导式设置初始化

init 命令会创建：

  • 配置文件 ~/.localbrain/config.yaml
  • 数据目录 ~/.knowledge-base/
  • 用于元数据的 SQLite 数据库
  • 用于向量嵌入的 ChromaDB 存储

配置 AI 服务（可选）

如需智能标签提取和语义搜索，请配置 AI 提供商：

localbrain config show  # 查看当前配置

编辑 ~/.localbrain/config.yaml：

embedding:
  provider: dashscope
  dashscope:
    model: text-embedding-v4
    api_key: your-api-key-here
llm:
  provider: dashscope
  model: qwen3.5-plus
  api_key: your-api-key-here

测试连接：

localbrain test embedding
localbrain test llm
ℹ️ 信息

AI 服务是可选的。即使没有它们，Local Brain 仍然可以使用内置的 TF-IDF 和提取式算法正常工作。
